This module tests the say my name function

>>> say_my_name = __import__('3-say_my_name').say_my_name

If you pass empty strings no name will be printed:
>>> say_my_name("", "")
My name is  

If you supply one name, only it will be printed
>>> say_my_name("Davies")
My name is Davies
